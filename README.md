# Daily Culinary

A small installable web app that shows one culinary lesson per day.

## Publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload all files from this folder to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.
6. Open the Pages URL after deployment finishes.

## Add to your phone Home Screen

### iPhone
Open the site in Safari, tap Share, then **Add to Home Screen**.

### Android
Open the site in Chrome, open the browser menu, then choose **Install app** or **Add to Home screen**.

## Edit lessons

Open `app.js` and add or change items in the `lessons` array.

The app stores saved and learned lessons in the browser with `localStorage`.
